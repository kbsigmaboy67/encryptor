▸ SECURITY NOTES
All cryptographic operations run entirely in your browser via the Web Crypto API.
No data is ever sent to any server. No cookies, no tracking, no logs.
The .vlt format: 4B magic + 12B IV + 32B salt + AES-GCM ciphertext + 16B auth tag.
Stego encoding: 256 invisible codepoints — Tags block U+E0000, variation selectors, zero-width marks.
Cover text and payload are separated by U+E007F TAG DELETE (zero visible width).
Content larger than 20 MB is not rendered in-browser — downloaded directly instead.
All operations local & offline AES-256-GCM · Web Crypto API · ChromeOS optimized
